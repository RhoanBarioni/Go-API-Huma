# 🌐 Guia de Operações CRUD e Verbos HTTP

## 📌 Mapeamento de Métodos

| Ação | Verbo HTTP | Descrição | Idempotente |
| :--- | :--- | :--- | :--- |
| **Create** (Criar) | `POST` | Cria um novo recurso. | Não |
| **Read** (Ler) | `GET` | Retorna a representação de um recurso. | Sim |
| **Update** (Atualizar) | `PUT` | Substitui um recurso existente na totalidade. | Sim |
| **Update** (Atualizar) | `PATCH` | Atualiza apenas partes específicas de um recurso. | Não |
| **Delete** (Remover) | `DELETE` | Elimina um recurso específico. | Sim |
